| Оператор | Пример использования | Эквивалент   |
| -------- | -------------------- | ------------ |
| `+=`     | `x += 5`             | `x = x + 5`  |
| `-=`     | `x -= 2`             | `x = x - 2`  |
| `*=`     | `x *= 10`            | `x = x * 10` |
| `/=`     | `x /= 4`             | `x = x / 4`  |
| `//=`    | `x //= 4`            | `x = x // 4` |
| `%=`     | `x %= 4`             | `x = x % 4`  |
